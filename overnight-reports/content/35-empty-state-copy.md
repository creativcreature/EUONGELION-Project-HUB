# EUONGELION Empty State Copy

*Friendly, sacred-toned messages for empty states throughout the app*

---

## Philosophy for Empty States

Empty states are opportunities, not failures. Each one should:
1. Acknowledge the current state without judgment
2. Gently invite action
3. Maintain the calm, sacred tone
4. Avoid creating anxiety or guilt

## Voice Guidelines for Empty States

### Do:
- Use gentle, inviting language
- Acknowledge without judging
- Offer a clear next step
- Keep it brief
- Maintain the sacred tone

### Don't:
- Make the user feel bad
- Use exclamation points
- Create urgency or guilt
- Be overly playful
- Use religious cliches

### Tone Examples:

| Avoid | Prefer |
|-------|--------|
| "You haven't done anything!" | "A blank page awaits." |
| "Don't forget to read today!" | "Your devotional is ready when you are." |
| "Bookmarks are empty :(" | "Nothing saved yet." |
| "Complete your first series!" | "Every journey begins with the first step." |

---

*Voice note: Empty states should feel like quiet invitations, not scolding parents. The user should feel welcomed back, not guilted for being away. EUONGELION's empty states are opportunities for grace, not performance tracking.*
